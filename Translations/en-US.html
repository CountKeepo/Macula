Macula Dark Theme This is a modification of the original Dracula Firefox theme based in the Dracula color palette. Modded Mac Dracula, aka Macula for Firefox I originally used the amazing Dracula theme on Windows, but after swapping over to Mac I found the default Dracula theme a bit lacking. So I went ahead and modified a version which better suited my needs. About Dracula: Dracula is a project created by Zeno Rocha with the help of many awesome contributors and maintainers. The dracula theme is also available officially for other platforms such as: Visual Studio Code, Atom, Telegram and many more. You can find them here. For unofficial themes you can check out Dracula's github If you would like to help, contribute to the project on Github.
